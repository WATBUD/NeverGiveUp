節次
動畫開始到結束，定義為一節次
無限循環(彩帶類)的動畫，視作無限久結束，總共一節次
 1. 從左界跑到右界
 2. 轉一圈    angle > 360
 3. 抵達外界  position > region.width
 4. 輪播結束  color_num * bandwidth_per_frame
 5. 無限反彈？ bump == 0

節次時間aka節次幀數
時間定義為幀數。每個效果都要提供一個節次的幀數
 1.彩帶類: a.節次時間無限大                          (時間決定距離)
           b.以bandwidth * color_num決定節次時間     (距離決定時間)
 2.波動類: a.自行指定節次時間                        (時間決定距離)
           b.region.width / bandwidth 有小數點，麻煩 (距離決定時間)

連射屬性
 1.單發: 場上只有一節次(彩帶類皆如此)   CD時間等於節次時間/播放時間
 2.連射: 場上同時出現許多節次           CD時間是參數
